- 협업·배포에 필요한 기본 파일 생성: `.gitignore`, `.env.example`, `README.md`
- **백엔드(`home-items-api`) uv 프로젝트** 뼈대 생성 및 동작 확인

### 주요 백엔드 의존성 (pyproject.toml)
- 런타임: fastapi, uvicorn[standard], pydantic, pydantic-settings, sqlalchemy, alembic, psycopg[binary], python-jose[cryptography], passlib[bcrypt], python-multipart
- 개발: ruff, pytest, httpx
- 도구 설정: ruff(line-length 100, py313, 규칙 E/F/I/UP/B), pytest(testpaths=tests)

## 4. 실행한 명령어

```bash
# 백엔드 의존성 설치
cd home-items-api
uv sync

# 코드 검사 + 최소 앱 동작 확인
uv run ruff check .
uv run python -c "from fastapi.testclient import TestClient; from app.main import app; \
print(TestClient(app).get('/health').json())"

# 독립 Git 저장소 초기화 + 원격 연결
cd ..
git init -b main
git remote add origin https://github.com/SeungIlJang/house-item.git
```
