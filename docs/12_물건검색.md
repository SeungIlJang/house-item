---

## 2. 설계 설명

| 구성 | 내용 |
|------|------|
| 키워드 | `name/description/memo` 에 `ILIKE '%키워드%'` (대소문자 무시) |
| 필터 | home_id, room_id, storage_location_id, category_id, tag_id 조합 |
| 태그 필터 | `item_tags` 조인 + distinct |
| 정렬 | `sort` = newest(기본)/oldest/name/-name |
| 페이지 | page/size(1~100) → PageData |
| 쿼리 파라미터 | 스펙대로 **snake_case** (`?room_id=1&category_id=2`) |
| 소유권 | 항상 `user_id` 조건 포함(내 물건만) |

> 참고: JSON 본문 필드는 camelCase 지만, 검색 **쿼리 파라미터**는 마스터 스펙에 따라 snake_case 를 사용합니다.
