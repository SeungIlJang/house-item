## 1. 이번 단계의 목표

- StorageLocation CRUD
- **계층형 구조**(parent_id): 붙박이장 > 오른쪽 칸 > 두 번째 서랍
- **전체 경로**(fullPath) 생성
- **순환 참조 방지**(자신/자손을 상위로 지정 금지)
- **트리 조회** API
- 테스트

---

## 2. 설계 설명

| 구성 | 내용 |
|------|------|
| 자기참조 | `parent_id` FK(self), 루트는 `parent_id = NULL` |
| 소유권 | Room → Home → user 로 판단 |
| 순환 방지(생성) | 새 노드는 자식이 없으므로 부모가 같은 방인지만 검사 |
| 순환 방지(수정) | 새 부모가 자신이거나 자손이면 409 `STORAGE_CYCLE` |
| 자손 계산 | 같은 방 전체를 읽어 parent 맵으로 BFS |
| fullPath | 부모를 따라 올라가며 이름 수집 + 방 이름 prefix |
| 삭제 | 부모 삭제 시 자식 CASCADE (FK ondelete + relationship) |
| 트리 | 평면 목록을 parent_id 기준으로 중첩 구조로 변환 |
